Project Introduction

This is a simple TaskTracker app which allows to create, edit, and track tasks with deadlines. Each user has his/her own JWC token as well as refresh token. Also, all passwords are hashed, so no worries about your privacy.

The backend is built with FastAPI, a high-performance framework for building APIs with Python. For data storage, I use PostgreSQL. The frontend is built with HTML and JavaScript for interactive features.

For database migrations, I use Alembic. As a package manager, the project relies on uv. The application is containerized and orchestrated with Docker Compose, which simplifies setup and deployment. Additionally, Nginx is used as a reverse proxy and to serve static files efficiently.
